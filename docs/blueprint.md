# **App Name**: KunalDeo.dev

## Core Features:

- Home: A brief professional introduction highlighting curiosity, learning mindset, and technical growth.
- Education: Display academic details (Second Year, Bachelor’s degree – Computer Science/IT).
- Skills: Showcase technical skills: Programming (C, C++, Java, Python), Data Structures & Algorithms, Operating Systems, Computer Organization, DBMS, basics of Cloud Computing (AWS), and Git/GitHub.
- Certifications: List certifications & training: EduSkills programs (AWS Cloud Foundation – in progress/completed), Google AI-ML Virtual Internship.
- Projects: Display academic or self-learning projects (beginner to intermediate level).
- About Me: Detail achievements & strengths: problem-solving, consistency, quick learner.
- Career Objective: State the career objective: To gain real-world industry exposure through internships and projects and build a strong foundation as a software engineer.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to convey intellect and focus.
- Background color: Light gray (#F0F0F0), very slightly tinted indigo, for a clean, professional feel.
- Accent color: Violet (#8F00FF) to highlight key sections and CTAs.
- Headline font: 'Space Grotesk', a sans-serif for a modern, techy look.
- Body font: 'Inter', a sans-serif to ensure readability.
- Use simple, line-based icons to represent skills and technologies.
- Subtle transitions and hover effects to provide feedback without being distracting.
